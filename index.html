Gov Icons is a font and CSS toolkit with 80 government themed icons, ready to be dropped right into any project that needs scalable vector icons that can instantly be customized!

CSS Control

Use the power of CSS to manipulate the color, size, showdow, etc. of any font icon. Anything that's possible with in CSS, you can do with govicons.

Use In Graphic Apps

Gov Icons come with OTF and TTF files, so you can use this page to copy icons and paste them right into your favorite design apps like Photoshop, Illustrator, Sketch!

Less and Sass Support

Originally designed for Bootstrap, Gov Icons works great with all frameworks. Use Less or Sass preprocessors to integrate the Weather Icons right into your existing builds.

Getting Started

To use Gov Icons, place the main CSS files in your CSS directory, and the font files in a "font" directory on the same folder level as the CSS director. Once you've done that, all you need to do to reference an icon in your HTML is type <i class="gi gi-tank"></i>
